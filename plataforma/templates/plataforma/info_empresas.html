Información para empresas

Descripción general del proceso de vinculación mediante SGVA.

¿Qué ofrece SGVA?

Un flujo centralizado para registrar información, recibir postulaciones y hacer seguimiento del proceso de selección de aprendices.

Postulaciones

Las postulaciones se gestionan con estados para facilitar el seguimiento (pendiente, seleccionado, contratado, etc.).

Acceso

La gestión interna está restringida a funcionarios SENA autorizados. Estas páginas son informativas.
